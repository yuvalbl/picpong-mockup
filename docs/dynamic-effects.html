Picpong · Motion & Interaction Library

Dynamic effects, ready to pick for the new site.

A working catalog of 18 tasteful, modern web interactions — each one demoed live with real Picpong project imagery. Pick the ones that fit; every effect is built on the brand tokens and the house easing curve cubic-bezier(0.62, 0.37, 0.1, 0.95), the same language that makes references like cartonlab.com feel calm and deliberate rather than busy.

Accessibility: every effect respects prefers-reduced-motion. With that OS setting on, animations resolve instantly to their finished state — content is never hidden behind motion, nothing loops, and the page stays fully usable.
